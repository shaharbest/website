## ceil

(like ceiling)

```javascript
Math.ceil(7.9); // 8
Math.ceil(7.5); // 8
Math.ceil(7.1); // 8
Math.ceil(7); // 7
```

## floor

```javascript
Math.floor(7.9); // 7
Math.floor(7.5); // 7
Math.floor(7.1); // 7
Math.floor(7); // 7
```

## round

```javascript
Math.round(7.9); // 8
Math.round(7.5); // 8
Math.round(7.1); // 7
Math.round(7); // 7
```

## random

```javascript
Math.random(); // 0.783724219048614
Math.random(); // 0.11949483671301542
Math.random(); // 0.2671735701764403
```

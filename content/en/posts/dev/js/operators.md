# Arithmetics

| Operator  | Result     |
| :-------: | :--------: |
| `+`       | sum        |
| `-`       | substract  |
| `*`       | multiply   |
| `/`       | divide     |
| `%`       | modulo     |
| `**`      | power      |

```js
let a1 = 1 + 1; // 2
let a2 = 10 - 5; // 5
let a3 = 2 * 2; // 4
let a4 = 6 / 3; // 2
let a5 = 5 % 2; // 1
let a6 = 2 ** 3; // 8
```

## Shorthand

| Concise   | Longer Equivalent |
| :-------: | :---------:       |
| `x += 2`  | `x = x + 2`       |
| `x -= 3`  | `x = x - 3`       |
| `x *= 4`  | `x = x * 4`       |
| `x /= 5`  | `x = x / 5`       |
| `x++`     | `x = x + 1`       |
| `x--`     | `x = x - 1`       |


# Logic Operators

* `&&` - AND
* `||` - OR
* `!` - NOT
